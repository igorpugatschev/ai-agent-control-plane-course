# AI Agent Control Plane Engineering

Самодостаточный русскоязычный курс по проектированию воспроизводимых AI-agent
workflow для разработки, QA и SDET.

## Первый шаг

**[Откройте учебный маршрут](curriculum/README.md)**, проверьте prerequisites и
начните с урока 1. Обязательный путь работает локально и не требует API-ключа.

## Практические материалы

- [Учебный Python-стенд](projects/training-task-app/README.md) - общий объект
  практики для модулей 2, 4, 5 и 6.
- [Шаблоны](templates/README.md) - заполняйте их по мере появления артефактов.
- [Контракты ролей](agents/README.md) - используйте после начала Module 3.
- [Checkpoint и финальная оценка](assessments/README.md) - открывайте после
  завершения соответствующего модуля.
- [Capstone](projects/capstone.md) - переходите к нему после Module 6.
