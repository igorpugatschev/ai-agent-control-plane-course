# Agents

В этой директории будут храниться учебные роли агентов и правила их взаимодействия.

Агент в рамках курса - это не “отдельная нейросеть”, а инженерная роль поверх LLM:

- с понятной задачей;
- с ограничениями;
- с входными данными;
- с ожидаемым результатом;
- с правилами остановки;
- с критериями качества.

## Планируемые роли

- coordinator agent;
- research agent;
- implementation agent;
- review agent;
- QA/SDET agent;
- documentation agent;
- risk and safety reviewer.

## Будущие артефакты

- шаблон роли агента;
- шаблон handoff;
- карта взаимодействия агентов;
- примеры рабочих инструкций;
- критерии качества агентного результата.
