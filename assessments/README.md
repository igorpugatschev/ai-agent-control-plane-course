# Assessment package

Этот пакет задает единый способ оценивать checkpoint и финальную защиту. Он не
является answer key: reviewer оценивает reasoning, наблюдаемое evidence, границы
решения и correction route, а не совпадение слов с образцом.

Используйте [guides checkpoint](checkpoints/README.md) после каждого модуля и
[финальную rubric](final-rubric.md) только для capstone. Перед оцениванием
проверьте artifacts и воспроизведите указанную command.

[Answer guidance](answer-guidelines.md) описывает quality patterns и
misconceptions, но не содержит готовых ответов на задания.
