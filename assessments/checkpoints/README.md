# Checkpoint assessment guides

Каждый guide содержит четыре критерия от 0 до 2, максимум 8. Для checkpoint
нужно не менее 6/8 и отсутствие critical defect; более строгий порог локального
checkpoint имеет приоритет. Эквивалентная формулировка допустима, если evidence,
roles, boundaries и recovery остаются проверяемыми.

Critical defect нельзя компенсировать баллами: reviewer называет artifact,
выполняет correction route и повторяет команду.
