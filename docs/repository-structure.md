# Карта репозитория

Начните с [учебного маршрута](../curriculum/README.md). Он связывает уроки,
накопленные artifacts и семь checkpoint.

| Место | Следующее действие |
| --- | --- |
| [curriculum/](../curriculum/README.md) | Пройти уроки по порядку и после модуля открыть его checkpoint. |
| [projects/](../projects/README.md) | Запустить training stand, затем заполнить starter и capstone. |
| [templates/](../templates/README.md) | Взять форму текущего artifact и сохранить её в student package. |
| [agents/](../agents/README.md) | Применить role contracts начиная с Module 3. |
| [assessments/](../assessments/README.md) | Оценить checkpoint или финальную защиту по evidence. |
| [glossary/](../glossary/terms.md) | Уточнить термин, не подменяя им практику. |

Документы в этой директории дают ориентиры: [миссия](course-mission.md),
[source policy](source-policy.md), [каталог источников](source-catalog.md) и
[матрица уроков и источников](lesson-source-matrix.md). Они не заменяют
локальные теорию, практику, correction route и проверки из уроков.
