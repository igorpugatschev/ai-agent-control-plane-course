# Требования к учебному стенду задач

## Назначение

Нужен детерминированный in-memory сервис задач для учебных упражнений по границам задачи, тестированию, анализу дефектов и проверке документации.

## Среда

- Python 3.11+;
- исходный код использует только стандартную библиотеку;
- для запуска тестов устанавливается `pytest`;
- сервис не требует сети, переменных окружения, базы данных или файлового хранилища.

## Модель

Задача содержит:

- целочисленный `id`;
- непустой `title`;
- `status` со значением `open` или `done`.

Новая задача получает статус `open`. Внутри Python это `TaskStatus.OPEN`, а завершённая задача имеет `TaskStatus.DONE`.

## Операции

### Создание

`TaskService.create_task(title)`:

1. удаляет пробелы в начале и конце названия;
2. отклоняет пустое или состоящее только из пробелов название с `ValueError` и сообщением `Название задачи не может быть пустым`;
3. отклоняет совпадение с названием другой активной задачи без учёта регистра с `ValueError` и сообщением `Активная задача с таким названием уже существует`;
4. выдаёт последовательный целочисленный идентификатор, начиная с `1`;
5. после завершения предыдущей задачи позволяет использовать её название снова;
6. возвращает созданную задачу.

### Получение списка

`TaskService.list_tasks()` возвращает все задачи текущего экземпляра сервиса в порядке возрастания `id`. В список входят и открытые, и завершённые задачи.

### Завершение

`TaskService.complete_task(task_id)`:

- для существующего идентификатора меняет статус на `done` и возвращает эту задачу;
- для неизвестного идентификатора поднимает `KeyError` с сообщением `Задача {task_id} не найдена`.

## Границы

Сервис не обязан запускать HTTP-сервер, сохранять данные после завершения процесса, аутентифицировать пользователей, удалять задачи или поддерживать параллельную запись. HTTP-представление тех же операций описано в [`api/openapi.yaml`](api/openapi.yaml).

## Проверка готовности

Готовность подтверждается командой:

```bash
PYTHONPATH=projects/training-task-app/src python3 -m pytest projects/training-task-app/tests -q
```

Все одиннадцать тестов должны пройти без предупреждений и ошибок импорта.
