# Templates

В этой директории будут храниться переиспользуемые шаблоны курса.

Шаблоны нужны, чтобы студент учился не только рассуждать об агентных процессах, но и оформлять их в воспроизводимом виде.

Планируемые шаблоны:

- agent role template;
- skill template;
- workflow template;
- review-gate template;
- stop-gate template;
- context map template;
- project blueprint template;
- final report template.

Каждый шаблон должен быть пригоден для копирования в реальный инженерный проект.
